# night-lamp-radio-Bluetooth-project
this is an integration of a night lamp with a bluettoth module that supports both bluetooth connectivity and radio
i also added a rechargeable battery that i got from an old laptop battery pack
for the speakers i used old toshiba laptop speakers and the sound is just good for nighttime.
this project was inspired by the desire to follow up on worldcup 2026 matches over the radio.
used it for a while and it is very reliable and discrete.
find the images documenting the whole process in the file above.
